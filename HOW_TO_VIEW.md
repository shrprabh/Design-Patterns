# How To View This Repo

This repo is meant to be read and run in a terminal.

## Quick view
- Open `README.md` for the kid-simple explanations and pattern list.
- Browse the code examples in `python/patterns/` and `csharp/Patterns/`.

## Run the demos

Python (3.10+):

```
cd python
python main.py adapter
python main.py all
```

C# (.NET 8):

```
dotnet run --project csharp -- adapter
dotnet run --project csharp -- all
```

## Notes
- Each run prints the demo output directly in the terminal.
- Pattern keys are listed in `README.md`.
